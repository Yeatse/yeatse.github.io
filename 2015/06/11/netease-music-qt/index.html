网易云音乐Qt版

Posted on 2015-06-11 | Edited on 2018-08-20 | In 笔记

使用Qt编写的非官方网易云音乐客户端，不定期更新。

最新版本

Version Download Release Note
0.9.6 Symbian / Harmattan - 修正了一处可能导致程序崩溃的bug。

历史版本

Version Download Release Notes
0.9.5 Symbian / Harmattan - 修复了部分歌曲无法加载封面图片的问题;
- (symbian)修正了qt4.8.0以前的机型音量控制不正常的问题。
0.9.4 Symbian / Harmattan - 支持乱序播放和单曲循环;
- 修复部分歌词解析错误的问题;
- 多处细节优化和bug修复。
0.9.3 Symbian - 支持订阅和取消订阅歌单;
- 支持查看歌单评论;
- 优化下载功能，音乐列表长按项目可直接下载;
- 支持显示歌词;
- 多处细节优化和bug修复。
0.9.2 Symbian - 新增搜索功能;
- 支持查看评论;
- 支持定时关闭程序;
- 可以修改下载保存的目录了。
0.9.1 Symbian - 增加下载功能;
- 多处细节优化和bug修复。
0.9.0 - 公开发布的初始版本。

代码已在GitHub上开源。
